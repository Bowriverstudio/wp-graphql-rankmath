=== Description ===

WPGraphQL Clarity is a free, open-source WordPress plugin that extends the GraphQL schema to include Clarity Project Id.

=== Dependencies ===
WP GraphQL
https://wordpress.org/plugins/microsoft-clarity/


=== GraphQL ===

```graphql
query MyQuery {
  claritySettings {
    projectId
  }
}
```
